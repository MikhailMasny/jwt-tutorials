## Getting Started

- Install `Node.js`;
- To configure the environment, the following commands were executed:

```
npm init
npm install eslint --save-dev 
./node_modules/.bin/eslint --init
npx install-peerdeps --dev eslint-config-airbnb-base
npm i bcrypt
npm i cookie-parser
npm i cors
npm i dotenv
npm i express
npm i jsonwebtoken
npm nodemon --save-dev
```
